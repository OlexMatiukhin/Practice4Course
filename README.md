# 🇺🇦 Українська версія

## 🔧 Загальна інформація

Програма призначена для генерації спеціальних файлів у навчальних цілях.

- ОС: **Windows**

---

## 📥 Встановлення

1. Встановіть Python 3.12.5 (Windows 64-bit)  
2. Завантажте `ProjectUtilities.exe`

### ⚠️ Попередження під час завантаження

![Download Warning](screenshots/download_warning.png)

- Натисніть **"Зберегти"**
- Якщо з'являється **"Видалити"** → змініть на **"Зберегти"**

3. Перемістіть файл у зручну теку  
4. Запустіть файл

![Run Warning](screenshots/run_warning.png)

- Натисніть **"Докладніше" → "Виконати"**

---

## 🧩 Інтерфейс програми

![Interface](screenshots/interface.png)

Доступні вкладки:

- ⚠️ **Створити Exe** — fork-бомба  
- ⚠️ **Створити PNG бомбу** — PNG-бомба  

---

## 📦 Створення файлів

### 🖥 Exe файл

![Exe Tab](screenshots/exe_tab.png)

1. Введіть ім'я  
2. Оберіть іконку:
   - Вбудовану  
   - Власну (.ico)  
3. Натисніть **"Запустити"**

![Generation](screenshots/generation.png)

📁 Файл створюється у папці `dist`

---

### 🖼 PNG-бомба

![PNG Tab](screenshots/png_tab.png)

1. (Опційно) змініть ім'я  
2. Натисніть **"Запустити"**

---

## ⚠️ Попередження

> ❗ НЕ запускайте згенеровані файли на власному ПК

---

## 🆘 Дії при помилці

### Exe

- Ctrl + Alt + Delete → Диспетчер задач  
- Завершити процес  
- Перезавантажити ПК  
- Видалити файл  

### PNG

- Перезавантажити систему
